# PES Version Control System (Mini Git)

## 📌 Overview
This project implements a lightweight version control system called **PES (Portable Efficient SCM)**, inspired by Git. It supports object storage, tree construction, staging (index), commit creation, and log history.

---

## ⚙️ Features
- Object storage using SHA-256 hashing
- Tree structure for directory snapshots
- Index (staging area) for tracking files
- Commit system with history tracking
- Log functionality to view commit history

---

## 🛠️ Technologies Used
- C Programming
- OpenSSL (for SHA-256 hashing)
- Linux/Ubuntu environment
